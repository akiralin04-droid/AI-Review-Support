# AI Review Support（仮）

## サイト概要
### サイトテーマ
AIがレビュー文章の作成をサポートするSNS型レビューサイトです。

### テーマを選んだ理由
私は読書をした際に内容を振り返るため感想を書こうとすることがありますが、文章を整理することに時間がかかり、投稿を面倒に感じてしまうことがありました。また、既存のレビューサービスでは投稿内容が人によってばらつきがあり、情報が十分に活用されていないと感じていました。

レビューはサービスや商品の評価、情報共有の手段として重要ですが、「何を書けばよいかわからない」「文章を考えるのが大変」といった理由から投稿のハードルが高いという課題があると考えています。この課題はレビューを書くユーザーだけでなく、レビュー情報を収集したい企業やサービス提供者にとっても共通の課題であると考えました。

そこで本アプリでは、ユーザーが短い感想を入力するだけでAIがレビュー文章を整理・生成する機能を実装し、レビュー投稿の負担を軽減することを目指しました。
さらに、単なる文章生成だけでなく、**「質問誘導による深掘り」「感情分析」「要約」**といったAI機能を活用することで、質の高いレビューを蓄積し、投稿者・閲覧者双方にとって価値のあるプラットフォームを実現したいと考え、このテーマを選びました。

なお、本アプリは読書レビューを題材に開発していますが、将来的には商品レビューやアンケート分析などにも応用できる設計を意識しています。

### ターゲットユーザ
・読書後に感想や学びを記録したいが、文章を書くことに負担を感じている人  
・レビューを投稿したいが、何を書けばよいか分からず投稿をためらっている人  
・他ユーザーのレビューを参考にしながら情報収集をしたい人  

### 主な利用シーン
・本を読み終えた後に感想や学びを簡単に記録したい時  
・読んだ本の評価や感想を他ユーザーと共有したい時  
・購入や読書の参考として他ユーザーのレビューを確認したい時  

## 実装予定の機能一覧
### AI機能（オリジナル機能）
OpenAI APIを活用し、以下の機能でユーザー体験を向上させます。
AIレビュー文章生成：断片的なキーワードから自然なレビュー文章を自動生成し、投稿のハードルを下げます。
AIレビュー質問誘導：AIからの質問に答える形式で、ユーザーの思考を整理しやすくします。
AIレビュー要約：長文のレビューを要約し、閲覧者が内容を短時間で把握できるようにします。
AI感情分析：レビュー内容からポジティブ・ネガティブなどの感情を分析し、可視化します。
レビュー分析ダッシュボード：投稿されたレビューの傾向などを分析・グラフ化し、全体のトレンドを把握できるようにします。

### ユーザー機能
会員登録・ログイン機能：メールアドレス・パスワードに加え、ゲストログイン機能（ワンクリックでお試し利用）を実装します。
マイページ：自分の投稿履歴や、他ユーザーからの「いいね」情報を確認できます。
ユーザー検索：質の高いレビュアーを探しやすくします。

## 投稿・レビュー機能
レビュー投稿：タイトル、本文、評価（★）、カテゴリに加え、画像投稿も可能です。
検索機能：
キーワード検索：気になるトピックを検索できます。
タグ検索：ジャンルやカテゴリごとにまとめてチェックできます。
一覧・詳細表示：投稿されたレビューを閲覧できます。

## コミュニケーション機能
いいね（ブックマーク）機能：共感したレビューを保存・評価できます。
コメント機能：レビューに対してリアクションや意見交換を行えます。

## 管理者機能
ユーザー管理：会員一覧の確認や、不適切な利用を行う会員の停止処理を行います。
レビュー管理：不適切なレビューを確認し、削除・非公開にする機能を提供します。
